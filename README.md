# Inteligencia Artificial 2025

Repositorio para subir las tareas del curso de Inteligencia Artificial 2025.

## Estructura del Repositorio

```
inteligencia_artificial_2025/
├── tareas/                    # Carpeta para las tareas del curso
│   └── tarea_01/             # Ejemplo de estructura de tarea
│       └── README.md         # Descripción de la tarea
├── proyectos/                # Carpeta para proyectos más grandes
├── recursos/                 # Materiales de referencia y recursos útiles
└── README.md                 # Este archivo
```

## Cómo Subir una Tarea

1. Crea una nueva carpeta dentro de `tareas/` con el formato `tarea_XX` (donde XX es el número de la tarea)
2. Incluye un archivo `README.md` con la descripción de la tarea
3. Agrega los archivos de código, documentación u otros entregables
4. Realiza un commit con un mensaje descriptivo

## Tecnologías

Este repositorio puede contener código en:
- Python
- Jupyter Notebooks
- Otros lenguajes según sea necesario
