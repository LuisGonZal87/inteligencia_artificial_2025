# Proyectos

Esta carpeta contiene los proyectos más grandes del curso de Inteligencia Artificial 2025.

## Estructura Sugerida

Cada proyecto debe tener su propia carpeta con:
- `README.md` - Descripción del proyecto
- `src/` - Código fuente
- `docs/` - Documentación adicional
- `data/` - Datos de entrada (si aplica)
